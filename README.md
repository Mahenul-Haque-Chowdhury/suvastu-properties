# Suvastu Properties

Premium real estate website for Suvastu Properties Ltd., built with Next.js, Tailwind CSS, GSAP, and Motion.

## Stack

- Next.js App Router
- React 19
- Tailwind CSS 4
- GSAP ScrollTrigger
- Motion

## Run Locally

Prerequisites: Node.js 20+

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Open `http://localhost:3000`

## Available Scripts

- `npm run dev` - start the local dev server
- `npm run build` - create a production build
- `npm run start` - run the production build
- `npm run lint` - run ESLint

## Project Notes

- Main marketing pages live in `app/`
- Shared UI and homepage sections live in `components/`
- Brand styling tokens are defined in `app/globals.css`
